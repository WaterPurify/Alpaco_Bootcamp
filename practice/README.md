

# Project 

* 기간 : 2023년 1월 ~ 2월
* 장소: 알파코 딥러닝 부트캠프
* 개요
 1. 웹과 딥러닝 두 부분을 진행했으며, 웹 부분은 플라스크를 사용
 2. 딥러닝은 CNN을 사용
* 데이터:  0~9의 숫자 이미지로 이루어진 MNIST train 6만장 test 3만장으로 사용
